# 宮本研面談予約システム

宮本研究室の面談予約システムです。ウェブブラウザで直接動作します。

## 使い方

1. フォルダ内の `index.html` を開いてください（ダブルクリックでブラウザが起動します）。
2. カレンダーが表示されます。
3. 日付をクリックすると予約画面が表示されます。
4. 情報を入力して「予約する」を押すと、予約が完了しカレンダー上の表示（空き状況）が更新されます。

## 機能
- **カレンダー表示**: 月ごとの予約状況がわかります（緑：余裕あり、黄：残りわずか、赤：満席）。
- **予約機能**: 名前、学籍番号、希望時間、内容を入力して予約できます。
- **データ保存**: 予約情報はブラウザの保存領域（LocalStorage）に保存されるため、ページを閉じてもデータは保持されます（同じブラウザである必要があります）。
